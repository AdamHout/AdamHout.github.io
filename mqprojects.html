Request / Reply

Platform: Linux
Language: C

This project demonstrates syncronous request and reply messaging between remote machines. Request messages consisting of a 64KB payload of random 32 bit unisigned integers are generated by program mqrequest on machine S1558. The requests are sent to remote queue DEV.Q1 via transmission queue E6410.TRANS.QUE over channel S1558.E6410.

Application program mqreply on machine E6410 pulls the requests of off DEV.Q1. It sorts the message payload data and sends a reply to remote queue DEV.Q2 via transmission queue S1558.TRANS.QUE over channel E6410.S1558.

Datagrams

Platform: Linux
Language: C

This project demonstrates the sending and receiving of datagrams via IBM MQ running on Ubuntu. Application mqdgram generates a payload of system resource utilization metrics at a rate of one message per second and places it on local queue DEV.ADAM. Application mqdgrec pulls the datagrams off of the queue and displays them on the console along with their message ID.
