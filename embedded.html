PS/2 Keyboard

Paltform: PIC24F MCU
Language: C

The PS/2 keyboard uses a two wire (clock and data) serial protocol that is relatively simple to interface compared to the complex USB protocol. This makes it an ideal choice as an input device for low power, bare metal MCU's. This project provides a library to process and translate the scan codes sent from the keyboard, which can then be sent to a display and/or be used to drive additional functionality.

EEPROM Libraries

Platform: PIC24F MCU
Language: C

Electronically Eraseable Programmable Read-Only Memory (EEPROM) is a form of non-volatile memory typically capable of greater than one million write cycles and data retention measured in hundreds of years. EEPROM's can be found in a wide variety of applications ranging from automotive to telecommunications. EEPROM's can also be emulated with flash memory that is enhanched to provide a greater number of write cycles and longer data retention than traditional flash. This project provides two sets of libraries as summarized below:

  1. The 24LC01B library provides an interface to Microchip MCP24LC01B family of EEPROM's via an I²C serial interface. This project includes code to demonstrate random access or sequential reads, byte and page writes, and object read and writes.

  2. The OBEEPROM library provides access to the on-board EEPROM (emulated) of a PIC24F16KA102 microcontroller. Included demo logic performs word and sequential reads and writes as well as word or bulk erase operations.

Digitial Potentiometer

Platform: PIC24
Language: C

A digital potentiometer also referred to as a "digipot", are intergrated circuits (IC's) that operates like its elctromechanical counterparts. Digipots are typically controled via a serial interface rather than the physcial movement of a wiper from a slide or shaft employed by traditional potentiometers. Due to the encapsulation of the IC, digipots are naturally more tolerant of environmental hazzards such as vibration, dust or grease that negatively impact mechanical pots. Digipots are used in a variety of applications ranging from analog control to medical devices.

This project provides a set of library functions to control an MCP41010 digipot from a PIC24F microcontroller over a Serial Peripheral Interface (SPI) connection and a demo driver to invoke them.
